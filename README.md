# Классификация Fashion-MNIST с использованием PyTorch

## Описание проекта

Данный проект посвящён решению задачи классификации изображений из набора данных Fashion-MNIST с использованием библиотеки PyTorch.

В рамках проекта проводится сравнение нескольких популярных архитектур сверточных нейронных сетей, а также исследуется влияние использования предобученных весов (Fine-Tuning) на качество классификации.

Проект реализован в формате production-like ML pipeline с использованием DataLoader, train loop, валидации, расчёта метрик качества и сохранения результатов экспериментов.

---

## Используемые архитектуры

В работе были исследованы следующие модели:

* AlexNet
* VGG16
* ResNet18
* EfficientNet-B0

Для каждой архитектуры были проведены два эксперимента:

### Scratch

Обучение модели с нуля на датасете Fashion-MNIST.

### Fine-Tuning

Использование предобученных весов ImageNet с последующим дообучением на Fashion-MNIST.

---

## Используемый датасет

Fashion-MNIST — это набор изображений предметов одежды, предназначенный для задач классификации изображений.

Состав датасета:

* 60 000 изображений для обучения;
* 10 000 изображений для тестирования.

Размер изображений:

28 × 28 пикселей.

---

1. T-shirt / Top
2. Trouser
3. Pullover
4. Dress
5. Coat
6. Sandal
7. Shirt
8. Sneaker
9. Bag
10. Ankle Boot

## Анализ результатов

По результатам экспериментов можно сделать следующие выводы:

1. Использование Fine-Tuning улучшает качество классификации для всех исследуемых архитектур.

2. Предобученные веса ImageNet позволяют моделям быстрее адаптироваться к новой задаче и достигать более высоких значений Accuracy и F1-score.

3. Наилучший результат показала модель EfficientNet-B0 после Fine-Tuning.

4. Среди моделей, обученных с нуля, наилучшее качество показала VGG16.

5. Разница между Scratch и Fine-Tuning особенно заметна для AlexNet и EfficientNet-B0.

## Заключение

В ходе выполнения проекта было проведено сравнение четырёх современных архитектур сверточных нейронных сетей на задаче классификации изображений Fashion-MNIST.

Было показано, что использование предобученных весов существенно улучшает качество моделей по сравнению с обучением с нуля. Наилучший результат продемонстрировала модель EfficientNet-B0 в режиме Fine-Tuning, достигнув значения F1-score 0.9388.

Полученные результаты подтверждают эффективность подхода Transfer Learning для решения задач компьютерного зрения.
