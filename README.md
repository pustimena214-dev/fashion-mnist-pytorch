# Классификация Fashion-MNIST с использованием PyTorch

## Описание проекта

Данный проект посвящен решению задачи классификации изображений из набора данных Fashion-MNIST с использованием библиотеки PyTorch.

В рамках проекта проводится сравнение нескольких популярных архитектур сверточных нейронных сетей.

## Используемые архитектуры

- AlexNet
- VGG16
- ResNet18
- EfficientNet-B0

## Режимы обучения

Для каждой архитектуры выполняются два эксперимента:

### Обучение с нуля (Scratch)

Все веса модели инициализируются случайным образом и обучаются только на данных Fashion-MNIST.

### Дообучение предобученной модели (Fine-Tuning)

Используются веса, предварительно обученные на наборе данных ImageNet, после чего модель дообучается на Fashion-MNIST.

## Используемый набор данных

Fashion-MNIST представляет собой набор изображений предметов одежды, состоящий из 10 классов.

Состав датасета:

- 60 000 изображений для обучения;
- 10 000 изображений для тестирования.
