# Классификация Fashion-MNIST с использованием PyTorch

## Описание проекта

Данный проект посвящен решению задачи классификации изображений из набора данных Fashion-MNIST с использованием библиотеки PyTorch.

В рамках проекта проводится сравнение нескольких популярных архитектур сверточных нейронных сетей.

## Используемые архитектуры

- AlexNet
- VGG16
- ResNet18
- EfficientNet-B0

## Режимы обучения

Для каждой архитектуры выполняются два эксперимента:

### Обучение с нуля (Scratch)

Все веса модели инициализируются случайным образом и обучаются только на данных Fashion-MNIST.

### Дообучение предобученной модели (Fine-Tuning)

Используются веса, предварительно обученные на наборе данных ImageNet, после чего модель дообучается на Fashion-MNIST.

## Используемый набор данных

Fashion-MNIST представляет собой набор изображений предметов одежды, состоящий из 10 классов.

Состав датасета:

- 60 000 изображений для обучения;
- 10 000 изображений для тестирования.

Размер исходного изображения:

```text
28 × 28 пикселей
Классы:

Футболка / Топ
Брюки
Свитер
Платье
Пальто
Сандалии
Рубашка
Кроссовки
Сумка
Ботинки

Этапы выполнения проекта

1. Разведочный анализ данных (EDA)

Выполняется:

анализ распределения классов;
визуализация примеров изображений;
проверка размеров данных.

2. Подготовка данных

Выполняются следующие преобразования:

изменение размера изображений до 224×224;
преобразование изображений в трехканальный формат;
нормализация данных;
создание DataLoader.

3. Обучение моделей

Для каждой архитектуры выполняются:
обучение;
валидация;
сохранение лучшей модели.

4. Оценка качества

Рассчитываются следующие метрики:
Accuracy;
Precision;
Recall;
F1-score.

5. Сравнение моделей

Полученные результаты собираются в единую таблицу для сравнения архитектур.

Вывод

В рамках проекта проводится сравнение современных сверточных нейронных сетей на задаче классификации изображений Fashion-MNIST. Выполняется анализ влияния предобученных весов на качество моделей и определяется наиболее эффективная архитектура для рассматриваемой задачи.
